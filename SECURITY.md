# Security policy

If you discover a security or privacy issue in this repository, please email **info@sevres.org** with the details rather than opening a public GitHub issue.

We will acknowledge receipt within a reasonable timeframe and work with you on responsible disclosure.

## Scope

This policy covers the CLASP specification, schemas, and worked examples in this repository. For issues found in other Sèvres-project components (`clasp-web`, `norma`, `sevres-web`, `clasp-mcp`), please use the same email; we will route appropriately.

For supply-chain or build-related issues affecting how Sèvres infrastructure consumes this repository (CDN, Vercel deployments, submodule pinning), please flag those as well.
